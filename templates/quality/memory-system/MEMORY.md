## メモリの種別

- **user**: ユーザーの役割・目標・スキル・好み
- **feedback**: 応答スタイル・行動指針（修正・承認の両方を記録）
- **project**: 進行中の作業・意思決定・背景（相対日付は絶対日付に変換して保存）
- **reference**: 外部システムへのポインタ（Linear・Slack・Grafana等）
